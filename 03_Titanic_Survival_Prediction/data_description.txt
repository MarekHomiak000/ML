Survived = Target variable (label)
        0 → Did not survive
        1 → Survived
        This is what ML models try to predict

Pclass = Passenger class (socio-economic status)
        1 → First class
        2 → Second class
        3 → Third class

Name = Passenger’s full name (includes title like Mr, Mrs, Miss)

Sex = Passenger gender

Age = Passenger age in years

SibSp = Number of siblings or spouses aboard the Titanic
        0 → traveling alone
        1 → spouse or one sibling

Parch = Number of parents or children aboard
        Often combined with SibSp to form:
        FamilySize = SibSp + Parch + 1

Ticket = Ticket number
        Mixed format (letters + numbers)

Fare = Ticket fare paid

Cabin = Cabin number
        HasCabin (Yes / No)
        Deck extracted from first letter (A, B, C…)

Embarked = Port of embarkation
    C → Cherbourg
    Q → Queenstown
    S → Southampton
